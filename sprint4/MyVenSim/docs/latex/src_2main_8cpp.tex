Função principal de execução do simulador. 


\begin{DoxyItemize}
\item 1. Cria o objeto \doxylink{classModel}{Model} principal (Fabrica).
\item 2. Cria e inicializa o \doxylink{classSystem}{System} "{}parafusos"{} com valor inicial de 1000.
\item 3. Cria o \doxylink{classFlow}{Flow} "{}venda"{}, configurando-\/o para subtrair 10 unidades do sistema "{}parafusos"{} a cada passo.
\item 4. Cria o \doxylink{classFlow}{Flow} "{}fabricacao"{}, configurando-\/o para adicionar 5\% do valor atual de "{}parafusos"{} a ele mesmo.
\item 5. Adiciona todos os componentes (\doxylink{classSystem}{System} e Flows) ao \doxylink{classModel}{Model}.
\item 6. Executa a simulação de tempo t=0 a t=100 com passo (dt) de 1.
\item 7. Imprime o valor final do \doxylink{classSystem}{System} "{}parafusos"{}.
\end{DoxyItemize}
