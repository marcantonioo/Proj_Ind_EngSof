\doxysubsection{Detailed Description}
Implementação da interface \doxylink{classImodel}{Imodel} que representa um modelo de sistemas e fluxos. 

A classe \doxylink{classModel}{Model} implementa os métodos definidos na interface \doxylink{classImodel}{Imodel}, permitindo adicionar e remover sistemas e fluxos, bem como executar a simulação do modelo ao longo do tempo.

\item {\ttfamily ritmo} (passo de tempo, ou dt) para cada iteração.
