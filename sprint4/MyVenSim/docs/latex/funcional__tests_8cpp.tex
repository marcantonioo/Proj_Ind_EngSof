\doxysubsection{Detailed Description}
Implementa os testes funcionais para validar a dinâmica do simulador (\doxylink{classModel}{Model}, \doxylink{classSystem}{System}, \doxylink{classFlow}{Flow}). 


\begin{DoxyItemize}
\item Os testes funcionais verificam se o sistema se comporta conforme esperado
\item para modelos de dinâmica de sistemas conhecidos, como Crescimento Exponencial
\item e Crescimento Logístico, além de testar a conservação de valor em modelos complexos. 
\end{DoxyItemize}
