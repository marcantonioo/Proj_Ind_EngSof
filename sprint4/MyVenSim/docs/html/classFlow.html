Detailed Description

Implementação da interface Iflow que representa um fluxo entre dois sistemas.

A classe Flow implementa os métodos definidos na interface Iflow, permitindo definir e obter o sistema de origem e destino, nome, equação do fluxo e avaliar o valor do fluxo com base na equação definida.
