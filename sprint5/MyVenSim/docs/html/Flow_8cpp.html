Detailed Description

Implementação da classe Flow que representa um fluxo entre dois sistemas.

A classe Flow implementa a interface Iflow e gerencia a origem (source) e o destino (target) do fluxo, bem como a equação que determina a taxa do fluxo com base no valor do sistema de origem.
