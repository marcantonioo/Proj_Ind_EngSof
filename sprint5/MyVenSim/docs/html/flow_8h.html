Detailed Description

Declaração da classe Flow que implementa a interface Iflow.

A classe Flow representa um fluxo entre dois sistemas (source e target) e inclui métodos para definir e obter esses sistemas, o nome do fluxo, a equação que determina a taxa do fluxo, e para avaliar o valor do fluxo.
